# Douyin Note And Image Fallback

Use this path when the final page is a note/image page or when no video media URL is available.

## Flow

1. Confirm that the link is not a playable video.
2. Read visible page metadata when available: title, account name, publish time, visible interaction counts, and image URLs.
3. Save or screenshot the visible image only when needed for analysis.
4. Analyze visible image/page evidence.
5. State clearly that no video understanding ran.

## Report Limits

Do not infer speech, captions, timing, cuts, or motion from an image page. Comments and engagement should be described only when visible or provided by the user.
